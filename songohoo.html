What is Songohoo?

Songohoo is a simple tool for creating and presenting song lyrics — designed for worship services, choir rehearsals, community singalongs, or any gathering where people sing together. It lets you build a presentation of songs that displays verse by verse on a big screen, so everyone can follow along without needing printed sheets.

Everything runs in a single HTML file, with no installation, no internet connection, and no external dependencies required.

🎵 Song Editor tab

Use this tab to create and manage your song files. Each song is saved as a small .json file on your computer.

Creating a new song

Fill in the optional Song Code (e.g. a number from your songbook), the Song Title, and then type each verse or refrain into the text areas below. A new text area appears automatically as you type into the last one.

Tick the Refrain checkbox on a section to mark it as a refrain. If the same refrain repeats after each verse, you only need to type its text once — for subsequent refrains, simply tick the checkbox and leave the text area empty. Songohoo will automatically reuse the first refrain's text when building the presentation.

Use the ✕ button to remove a section you no longer need.

Saving a song

Click 💾 Save Song. A save dialog will open — choose a folder and confirm. The filename is generated automatically from the code and title (e.g. 42_ShineJesusShine.json).

Editing an existing song

Click 📂 Open Song File, navigate to a previously saved .json song file, and select it. The song data will load into the editor. Make your changes, then save again.

🎤 Presentation tab

Use this tab to assemble a set of songs into a presentation and display it fullscreen.

Setting up

  • Presentation Title — shown at the top of the song list slide (e.g. "Sunday Service" or "Easter Worship").
  • Logo — optionally select an image (JPG, PNG, SVG) to display alongside the title on the song list slide.
  • Divider Colour — the colour of the line under the title and the dot indicator on the song list. Use the colour picker or type a hex code.
  • Max Lines / Slide — controls how many lines of lyrics can appear on one slide. If a verse and its refrain fit within this limit, they'll be shown together; otherwise the refrain gets its own slide.

Adding songs

For each row, click 📂 to select a song file. The code and title will populate automatically. Optionally add a short description (shown in the song list). A new row appears as you add songs. Use ✕ to remove a song from the list.

Starting the presentation

Click ▶ Start Presentation. The screen goes fullscreen black with white text. The presentation is structured as follows:

  • A Song List slide showing all songs, with a coloured dot indicating which song is next.
  • The verses and refrains of each song, one slide at a time.
  • Another Song List slide between each song (dot moves to the next song).
  • A final Song List slide at the end (no dot).

Saving the presentation

Click 💾 Save as HTML to export the entire presentation as a standalone HTML file. This file contains all song lyrics and the logo embedded inside it — you can copy it to any computer and open it in a browser to present, with no other files needed. The filename defaults to the presentation title.

Starting fresh

Click ✨ New Presentation to clear all songs and settings and start from scratch.
